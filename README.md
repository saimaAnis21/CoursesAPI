# Course - API
# [Live Link]()

A  ![](https://img.shields.io/badge/Microverse-blueviolet)  capstone project for React-Redux Module

A course API for enrolling students in courses

## Built With

- Rails
- Ruby
- Rspec
- rubocop
- Heroku

## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone this project by typing ```git clone git@github.com:saimaAnis21/CoursesAPI.git```
- cd into the root directory
- Run git pull origin main
- run ```bundle install``` to install the project dependencies
- run ```rails s``` to start the project

## Testing
- run ```bundle exec rspec``` to run the tests
